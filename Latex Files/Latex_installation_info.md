---------------------------------------------
Instructions for "compiling" latex references
---------------------------------------------
1. Go to References file. Compile based on bibtex.
2. Then go to Main file (Master Thesis En). Compile based on bibtex.
3. Then press F6,F11,F6 button.
4. Then do Main file (Master Thesis En) compile based "Γρήγορη Μεταγλώτιση"
5. The above process should be done 2 times.
